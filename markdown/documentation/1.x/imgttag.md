
<h3 class="display-4 mb-5">ImgTTag</h3>


**Category :** `class`

```php
namespace Src\TTags;
```

**Overview**

It creates the `<img>` tag.
	

**Usage Syntax**

```php
<?php 

// Namespace
use Src\TTags\{ImgTTag};

// Useage
new ImgTTag($src ,$attribute = null);

```
<p class = "ttag-code-caption text-muted"><b>ImgTTag.php</b></p>


**Parameters**

**`$src` : String**

Image's source address.

---

**`$attribute` : String**

Any attribute like `class`, `data-someData` etc.

**`null` :** <span class="badge badge-dark">Default</span> No attribute will be added.

---
